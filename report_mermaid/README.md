# Thesis Mermaid Figures

These Mermaid files replace the rough autogenerated PNG diagrams with cleaner diagram source you can:

- paste into Mermaid Live Editor
- render in Markdown-enabled tools
- export as SVG/PNG for the thesis

Recommended usage:

- Use Mermaid for architecture, workflow, protocol, and component diagrams.
- Use tables for final metrics.
- Use native ROC/PR images from `model/evaluation/...` for quantitative curves, since Mermaid is weak for scientific curve plotting.

Files:

- `architecture_system_overview.mmd`
- `enrollment_authentication_flow.mmd`
- `runtime_fusion_logic.mmd`
- `class_structure.mmd`
- `dataset_composition.mmd`
- `modality_performance_comparison.mmd`

- `training_dynamics.mmd`
